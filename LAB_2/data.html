What is AJAX?

AJAX stands for Asynchronous JavaScript and XML. It allows web pages to communicate with the server in the background and update parts of the page without a full page reload.

Key Points:

  • Uses XMLHttpRequest object
  • Works asynchronously (does not block the page)
  • Can load HTML, JSON, XML, or plain text
  • Updates only a part of the page, not the whole page
